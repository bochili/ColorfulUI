平滑滚动
该动画核心代码来自于https://www.cnblogs.com/sapho/p/6022829.html
锚点链接
就像上面的目录，在点击锚点链接，跳转时会触发动画，实现平滑的滚动效果。在链接上添加类.cui-animate-roll，再编写好id即可实现



code
<a href="#link1" class="cui-animate-roll">平滑滚动</a>
<div style="height: 300px;"></div>
<div id="link1">I AM HERE</div>



返回顶部
当在body标签上添加id，再通过置于页面下方的锚点链接，在链接上添加类.cui-animate-roll，即可实现返回顶部效果



code
<body id="top">
	Top
	<div style="height: 300px;"></div>
	<a href="#top" class="cui-animate-roll">返回顶部</a>
</body>



鼠标悬浮
Hover效果
许多CUI组件拥有Hover的CSS效果，如果您想在自定义组件上添加Hover效果，可以在该组件上添加类.cui-animate-hover，即可实现在该元素上的鼠标悬浮效果。

UP效果
您可以在某些自定义/非自定义组件上添加类.cui-animate-hover-up，即可实现在该元素上的鼠标悬浮上升效果。

阴影效果
您可以在某些自定义/非自定义组件上添加类.cui-animate-hover-shadow-[COLOR]-light或.cui-animate-hover-shadow-[COLOR]-[1-24]，即可实现在该元素上的鼠标悬浮阴影效果。
